## 2024-06-27 - Fix Stored XSS via unescaped albumArt **Vulnerability:** Stored XSS via unescaped track.albumArt string interpolation. **Learning:** DOM properties (e.g. `img.src`) should not use HTML escaping since they expect raw strings; `escapeHtml` should strictly be used within HTML concatenations. **Prevention:** Validate all string interpolations in HTML template strings and distinguish between HTML text content and element properties.
