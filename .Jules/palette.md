## 2024-05-22 - [Dynamic ARIA Attributes] **Learning:** When updating ARIA attributes dynamically via JavaScript (e.g., setAttribute('aria-expanded', value)), you must ensure the value is explicitly passed as a string ('true' or 'false'), rather than a boolean. Passing a boolean directly might be evaluated incorrectly or interpreted differently across browsers. Also, it is an accessibility best practice to link UI toggle buttons to their collapsible panels using aria-controls="[panel-id]". **Action:** Always add string values to dynamic aria attribute updates and link toggles using aria-controls.
