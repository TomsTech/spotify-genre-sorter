## 2024-05-17 - Sidebar Toggle Accessibility Fix **Learning:** When setting ARIA attributes dynamically in JS (like `aria-expanded`), the value must be explicitly cast to a string (e.g., `'true'` or `'false'`) rather than passing a boolean to ensure strict ARIA compliance and avoid linter or review errors. Furthermore, for toggle buttons that control collapsible panels, it's important to use `aria-controls` paired with `aria-expanded`. **Action:** When reviewing or writing DOM manipulation logic involving ARIA attributes, ensure boolean state variables are explicitly converted to string literals before being passed to `setAttribute()`.
